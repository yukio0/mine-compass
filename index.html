MineCompass - Minesweeper Solver

JA GitHub
MineCompass helps you analyze classic Minesweeper positions. Enter a board position to see the analysis.
For the best experience, use a desktop browser.
Enter covered cells, flags, and revealed numbers to highlight guaranteed safe cells, guaranteed mines, and the best candidate cells with the lowest mine probability.
Bug reports and feedback are welcome on GitHub.
Use the results as a helpful reference while solving.
Current tool:
Covered Flag Revealed (number or empty) Guaranteed safe Guaranteed mine Recommended (lowest mine probability)
